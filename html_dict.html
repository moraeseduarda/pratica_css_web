Dicionário de Elementos HTML

<!DOCTYPE html>
Define o tipo de documento como HTML5.
<html>
Elemento raiz de um documento HTML.
<head>
Contém metadados, título e links para estilos/scripts.
<body>
Contém o conteúdo visível da página.
<h1> ... <h6>
Definem títulos, do mais importante (h1) ao menos importante (h6).
<p>
Define um parágrafo de texto.
<a>
Cria um link para outra página ou recurso.
<nav>
Define uma seção de navegação com links.
